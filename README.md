# nvim-config

## Requirements

-   Neovim v0.9+
-   [Nerd Font](https://github.com/ryanoasis/nerd-fonts)
-   [fd](https://github.com/sharkdp/fd)
-   [ripgrep](https://github.com/BurntSushi/ripgrep)

## Setup

1. `mkdir -p ~/.config`
2. `cd ~/.config`
3. `git clone git@github.com:steschwa/nvim-config.git nvim`
